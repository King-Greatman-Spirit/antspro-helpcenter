Work Completion Workflow

  • Certification Initiation: Begin the process when work reaches 90% completion or project milestones are achieved.
  • Equipment Reconciliation: Verify all equipment used (cranes, trucks, tools) with actual deployment days.
  • Financial Calculation: System automatically computes day rates, standby charges, and VAT amounts.
  • Multi-Level Approval: Requires sign-off from vendor reps, department heads, and technical teams.

Key Certification Features

Automated Numbering

Unique WCC numbers (e.g., 8877) generated for each certification with vendor invoice tracking

Financial Automation

Calculates total days (base + extended), applies standby rates (e.g., 15.00.0.0), and computes VAT (7.5%)

Completion Metrics

Tracks percentage of work completed and off-hire dates (e.g., 2025-04-30) for equipment

Payment Processing

Generates net payable amounts (e.g., 38.750.0.0) with complete audit trail for finance

Work Completion Certification Process

The certification process involves multiple verification steps to ensure work quality and proper documentation:

Equipment & Resource Verification
  • Equipment Checklist:
    • Mark all deployed equipment (Crane, Low Bed, Forklift, etc.)
    • Specify "Others" for non-standard equipment
    • Record actual deployment days vs. planned
  • Vendor Details:
    • Select vendor code (e.g., VE-01) and name
    • Verify project initiator code (e.g., 61)
Financial Documentation
  • Day Rate Calculation:
    • Base days (e.g., 7) at standard rate (e.g., 15.00.0.0)
    • Extended days (e.g., 3) at standby rate
    • VAT computation (7.5% of total)
  • Payment Details:
    • System calculates net payable amount
    • Records evaluation date (e.g., 2025-04-30)

Certification Best Practices

  • Complete all equipment checkboxes accurately to prevent billing disputes
  • Verify percentage completion matches actual work performed
  • Ensure all dates (off-hire, evaluation) reflect true project timeline
  • Attach supporting documents before HOD submission
  • Double-check financial calculations against contract terms
