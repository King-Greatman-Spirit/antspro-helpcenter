W.C.C - Work Completion Certificates Help Center

This section of your AntsPro Dashboard is designed to help you track, manage, and follow up on all Work Completion Certificates (W.C.C.) within your project workflow — from preparation to approval and archival.

You Have Questions, We Have Answers

Navigating the Operations Dashboard

  • Dashboard: return to the main dashboard overview.
  • New WCC: Create a new W.C.C - Work Completion Certificates using Form 140.
  • Back: Go back to the previous page you visited.
  • Refresh: Reload the current page to view the latest updates.

Tabs Explained

Each tab on the page filters W.C.C - Work Completion Certificates records based on their current workflow stage:

  1. Archived: Displays certificates that have been fully processed and archived for reference. These require no further action and are kept for recordkeeping.
  2. Pending Approval: Shows W.C.Cs that have been submitted but are awaiting official approval from the relevant authority.
  3. Pending End User Signature: Lists W.C.Cs that are awaiting a signature from the designated end user before final processing can occur.
  4. Pending Submission: Contains W.C.Cs that have been prepared but are yet to be formally submitted into the approval process. These may require review or final edits.

Navigating and Using the Table Efficiently

To make the most of the Work Completion Certificates (W.C.C) table, the platform provides multiple user-friendly tools to help you find, view, and analyze records quickly:

  1. Search Input: Located above the top-right corner of the table, this feature allows you to instantly filter results by typing keywords such as WCC No., Vendor Name, Net Amount, or any relevant value. The results are updated in real time as you type.
  2. Records per Page Dropdown: Found on the top-left of the table, this dropdown allows you to control how many entries are visible at once. You can expand the view from 10 to 25, 50, or more records per page for broader visibility.
  3. Pagination Controls: Below the table, you’ll find numbered buttons (1, 2, 3…) that let you navigate between pages. Simply click on any page number or use “Previous” and “Next” to move through the records.
  4. Sortable Columns: Each table header has a small arrow icon. Clicking on these arrows allows you to sort the table data in either ascending or descending order based on that column (e.g., by Date, Net Amount, Vendor, etc.).

Pro Tip: These tools are designed to streamline your workflow, helping you access and manage large datasets with ease and precision.

Understanding the Table View

Each tab shows a table with the following key details:

  1. Date: The date the certificate was prepared.
  2. WCC No.: Unique ID for each Work Completion Certificate.
  3. Vendor: The company or contractor that carried out the work.
  4. Prep. By, Initiator/Lead, End User, Approval: Status indicators showing each person’s action (e.g., Pending, Signed, Approved).
  5. Net Amount: Total value of the work completed (may show error if data format is incorrect).
  6. Use View/Print for authorization/approval reference.
  7. Use View/Verify (Approve/View) for final review and approval steps.
  8. Use View/Verify (Sign/View) to review and sign documents.

Note: If you see a warning or error message (like a number_format() issue), it’s usually due to incorrect data input. Contact support for quick resolution.

Pro Tip: You can use the Search bar or Records per page dropdown to easily navigate large volumes of data. This helps you filter results quickly and improve your workflow efficiency.
