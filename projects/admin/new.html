Projects Reports Dashboard Help Center

The Projects Reports Dashboard provides tools for managing and accessing project reports. This guide explains the interface components and their functions.

You Have Questions, We Have Answers

Top Navigation Bar

  • Dashboard: Return to the main projects overview at any time.
  • Back: Returns to the previous page in your navigation history.
  • Refresh: Reloads the current dashboard to display the most recent data.

Projects Reports Dashboard Features

  • Select Report Icon: Choose from available report types including Book, List Report Under Process, Cert, and Report Code options.
  • Department: Use the dropdown to filter reports by department or select "Report Title" to view all available reports.
  • Report Link: Direct access to report documents or related resources.
  • Log Section: Displays creation information including the creator's name (greafjGreaman) and timestamp (24-04-2025 / 12:56:52).

Quick Tips

  • Use the Department dropdown to quickly filter reports by specific teams or units.
  • The Refresh button ensures you're viewing the most current report information.
  • Check the Log section to see when reports were last created or modified.
  • Different Report Icons help visually distinguish between report types.
