Admin Projects Help Center

The Projects area in ANTsPro II provides a centralized dashboard for managing and monitoring all project-related activities across departments. Use this guide to quickly understand the layout and how to navigate the Projects environment, especially handling specific project fields like Mobilization, Project Code, Supervisor details, and Client Logos.

You Have Questions, We Have Answers

Top Navigation Bar

  • Dashboard: Return to the main projects overview at any time.
  • New Project: Start new entries for project reports like NUN RIVER 7.
  • Back: Go back to the previous project page or form.
  • Refresh: Reload the project page to see the latest updates, such as added Mobilization or new project details.

Project Form and Fields Guide

  • Scope: Select whether the project is for Mobilization, Re-Mobilization, or both.
  • Date: Record the project start date (e.g., 2024-02-07 for NUN RIVER 7).
  • Project Code: Unique project code (e.g., PRJ000686).
  • Client Company: Enter the client's company name (e.g., GEOPLEX).
  • Department: Indicate the responsible department (e.g., Technical).
  • Location, District, Station: Provide specific location data (e.g., Delta, DELTA, NUN).
  • Terrain: Specify if it is Offshore, Onshore, etc.
  • Project Supervisor & Engineer: Assign key personnel (e.g., UWABOR JONAH).
  • Field and Type of Operation: Specify technical field and operation type (e.g., Pressure Control).
  • Client Logo: Upload the client logo to personalize reports.
  • Log: Track modifications (e.g., who updated the form and when, like Sakepere Emanor Priscilla on 07/02/2024).

Project Reports Dashboard Management

  • Reports Table: Displays project-specific details such as Scope, Project Code, Supervisor, and Client Company. Each row represents a project like NUN RIVER 7, with fields for monitoring progress.
  • Project Management Tip: Ensure that fields like MOB Date (Mobilization Date), DEMOB Date (Demobilization Date), and Status (e.g., PREPARATORY) are regularly updated for accurate project tracking.
  • Focal Person: Each project assigns a focal contact person (e.g., JOSHUA AKANLE) for easy communication.

Quick Tips

  • Use Ctrl + F (or Cmd + F on Mac) to quickly search for any Project Code, Supervisor Name, or Client Company on the dashboard.
  • Click Refresh to pull the latest form updates, such as changed Mobilization status or new remarks.
  • Use the Log field to maintain an audit trail of who created or modified project details.
  • Attach logos properly using the Choose File button for branding project reports.
