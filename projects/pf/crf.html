Cash Retirement Forms Management Center

This dashboard section helps you track, verify, and process all Cash Retirement Forms for the NUN RIVER 7 project - from receipt submission through final approval and filing.

Expense Accountability System

Navigating the Retirement Dashboard

  • Dashboard: return to the main dashboard overview.
  • New Retirement Form Create documentation for spent cash advances.
  • Back: Go back to the previous page you visited.
  • Refresh: Update to view latest retirement status changes.
  • Project Title: Shows current project (NUN RIVER 7) and department context.

Retirement Workflow Stages

Each tab filters retirement forms by their current accountability phase:

  1. Processed & Filed: Completed retirements with all receipts verified and permanently archived.
  2. Pending Processing: Submitted retirements awaiting initial finance department review.
  3. Pending Approval: Requires manager authorization after verification.
  4. Pending Pass: Needs secondary verification by accounts team.
  5. Pending Sign/Salient: Awaits final sign-off or requires special attention.

Retirement Forms Tracking

The table displays all pending retirement forms with these key details:

  1. Date: When the retirement was submitted.
  2. Form No: Unique retirement document identifier.
  3. By: Staff member submitting the retirement.
  4. Reference: Related documents or advance form number.
  5. Res. By: Responsible officer for verification.
  6. Pass Status: Secondary verification completion.
  7. Approval Status: Manager authorization state.
  8. Receipt/Payment Status: Whether supporting documents are complete.
  9. Amount (=N= : K): Total being retired (Naira/Kobo).

Retirement Best Practices

  • Submit retirements within 72 hours of expenditure
  • Include all original receipts with clear dates/amounts
  • Reference the original advance form number
  • Highlight any variances with explanations
