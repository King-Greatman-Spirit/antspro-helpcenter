Weekend-Work Allowance Management Center

This dashboard tracks all Weekend-Work Allowance Forms for the NUN RIVER 7 project - from submission through approval and payment processing.

You Have Questions, We Have Answers

Navigating the Allowance Dashboard

  • Project Title: Shows current project (NUN RIVER 7) and department context.
  • New Allowance Form Create weekend-work compensation requests.
  • Refresh: Update to view latest status changes.
  • Workflow Tabs: Filter forms by current processing stage.

Allowance Workflow Stages

Each status represents a different phase in the approval process:

  1. Issued & Archived: Completed allowances with all documentation verified and permanently filed.
  2. Pending Issue: Approved forms awaiting payroll processing.
  3. Pending Approval: Requires department head authorization.
  4. Pending Pass: Needs HR verification of work hours.
  5. Pending Submission: Draft forms not yet officially submitted.

Allowance Forms Tracking

  1. Date: When the weekend work occurred (YYYY-MM-DD format).
