Petty Cash Voucher Management

This section provides complete guidance for managing B.G. Technical Limited petty cash disbursements, including voucher preparation, approval workflows, and payment tracking for Pipeline and Well Services operations.

Cash Disbursement Documentation

Navigating the Budget Dashboard

  • Dashboard: Return to the main dashboard overview.
  • Back: Go back to the previous page you visited.
  • Refresh: Update to see the latest authorization status changes.

Navigating the PCV System

  • Company Header: Identifies the form as belonging to B.G. Technical Limited's Pipeline and Well Services division.
  • Voucher No: Unique identifier for each petty cash transaction (e.g., "PICNIR").
  • Project Reference: Links the disbursement to specific project accounting (e.g., "NUM IN 0767 / .06L").
  • Transaction Date: Documents when the voucher was created (e.g., "2025-05-22").

Financial Tracking

ACCOUNT TO CHARGE: Project account being debited (NUM IN 0617 / .06L)

AMOUNT IN WORDS: Required legal description of amount

TOTAL AMOUNT: Sum of all line items (auto-calculated)

Authorization Process

Requester Section

  • Prepared By: Staff member creating the voucher
  • Submit Date: When voucher was submitted (01/04/2025)
  • Submitted By: Name of submitter (Unpacked)

Approval Section

  • Approved By: Manager authorization
  • Received By: Who collected the cash
  • Signature Date: When signed (2025-05-22)

PCV Best Practices

  • Always obtain receipts for all disbursements
  • Record exact purpose of payment with details
  • Ensure dual authorization (preparer + approver)
  • Reconcile petty cash weekly
