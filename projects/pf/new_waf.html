Weekend-Work Allowance Form Center

Complete guide for preparing, submitting, and tracking Weekend-Work Allowance claims for B.G. Technical Limited's NUN RIVER 7 project in the Technical Department.

About Weekend-Work Allowances

Official compensation documents that:

  • Record weekend work hours and activities
  • Calculate approved compensation rates
  • Require supervisor verification
  • Generate payroll records

Forms must be submitted within 3 working days after weekend work completion.

Key Form Components

Header Information

  • To: Recipient name (Unye-Awaji Greatman)
  • Department: Technical (auto-filled)
  • Form No: WAL/05/25/10708 (auto-generated)
  • Project: NUN RIVER 7 (selected from dropdown)
  • Date: Work date (2025-05-05)

Time Tracking

  • S/No: Line item numbering (1-10)
  • Time In/Out: Work hours (--:-- if N/A)
  • Duration: Calculated work hours
  • Purpose: Work description
  • Amount: Calculated compensation

Weekend Allowance FAQs

For NUN RIVER 7 project:

  • Minimum 4 continuous hours on Saturday/Sunday
  • Approved emergency maintenance
  • Critical path project activities
  • Pre-approved by supervisor

Normal shift rotations don't qualify.

Position Base Rate Weekend Rate Example (8hrs)
Technician ₦750/hr ₦1,500/hr ₦12,000
Supervisor ₦1,250/hr ₦2,500/hr ₦20,000

Standard processing timeline:

  • Approved by Wed: Included in Friday payroll
  • Approved after Wed: Following week payroll
  • Emergency work: Special processing within 48hrs
Delays may occur if documentation is incomplete.
