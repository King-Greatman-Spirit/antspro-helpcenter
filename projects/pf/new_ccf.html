Cash Expense Claim Help Center

Complete guide for submitting Cash Expense Claims in B.G. Technical Limited's Pipeline and Well Services with specific focus on NUN RIVER 7 Project requirements.

About Cash Expense Claims

Official documents for reimbursement that:

  • Record all project-related expenditures
  • Attach original supporting documents
  • Specify exact cost allocation
  • Require proper authorization

Technical Department claims require dual verification for amounts above ₦500,000.

Header Section

  • Department: Technical (non-editable)
  • Project: NUN RIVER 7 (selected from dropdown)

Expense Table

  • SIGN: Verification checkbox
  • COST CODE: Project-specific GL codes
  • PARTICULARS: Detailed description
  • AMOUNT: ₦ value with 2 decimals

Technical Department FAQs

For NUN RIVER 7 fuel claims:

  1. Original pump receipt showing:
    • Station name/location
    • Date/time of purchase
    • Quantity in litres
    • Price per litre
  2. Equipment usage log signed by supervisor
  3. Daily consumption report
  4. GPS coordinates (for field operations)
