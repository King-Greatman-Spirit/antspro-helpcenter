Understanding the Equipment Lists View

The Equipment Lists Interface

This screenshot displays the comprehensive Equipment Lists view, which serves as the central hub for accessing, monitoring, and managing all registered equipment assets within the GEMRS system. From this powerful interface, you can efficiently browse through the complete inventory database, perform advanced searches using multiple criteria, filter results by equipment classification, access detailed specifications and documentation, and manage equipment allocation across different project sites and departments.

Interface Element Description How to Use
Records per page dropdown Controls how many equipment items are displayed per page Click the dropdown menu (showing "10") to select a different number of records to display (options typically include 10, 25, 50, 100)
Search field Allows filtering equipment by any visible column data Type keywords related to equipment number, classification, or description to instantly filter the list
Equip. No. column Displays the unique numerical identifier for each piece of equipment Click on the column header to sort the list by equipment number
CL column Shows the equipment classification or category Quickly identify equipment types (e.g., "Helium Leak", "Pressure Relief Valves", "Pumps", "Hoses")
Document icon column Indicates available documentation for each equipment item Click the PDF icon to view or download equipment documentation, specifications, and manuals
S/No. R/No. column Displays the Serial Number or Reference Number for each equipment Use these detailed identifiers (e.g., "BGT/PU/LD-005") when requesting specific equipment
Description column Provides detailed information about each equipment item Contains equipment names and specifications; click the list icons to view additional details or options
Pagination controls Navigate through multiple pages of equipment records Use the numbered buttons or "Previous"/"Next" links to move between pages of results

Equipment List Navigation Tips

  • Use specific search terms to quickly locate equipment by type, number, or description
  • The list displays your search results in pages of 10 items by default (showing 1 to 10 of 1,499 entries)
  • Click the PDF icon to access technical documentation for any equipment item
  • Equipment is organized by classification (CL) for easier browsing by category

For additional assistance with equipment lookups, contact the GEMRS support team.

GEMRS Best Practices

Form Completion

  • Verify all equipment numbers match asset tags
  • Include complete serial numbers for traceability
  • Specify exact FROM/TO locations (building/room)
  • Attach supporting documents if required

Authorization Process

  • Submit at least 24hrs before needed movement
  • Follow up with approvers via email/phone
  • Verify all signatures before dispatch
  • Keep digital/physical copies for audit

Special Handling Required For:

  • Calibrated instruments
  • Hazardous materials
  • High-value assets (>₦10M)
  • Customer-owned equipment
  • Temperature-sensitive items
  • Government-regulated assets
