Understanding the Equipment Lists View

The Equipment Lists Interface

This view displays all registered equipment assets in a paginated table format. The interface allows you to quickly search and filter through 1,499 equipment items, with 10 items displayed per page by default. Key equipment categories include Helium Leak Detectors, Pressure Relief Valves, Pumps, and various Hose types, each with detailed specifications and unique identifiers.

Interface Element Description How to Use
Records per page Controls pagination of equipment items (default: 10) Currently displays 10 items per page from total 1,499 entries
Search field Filters equipment by any column data Try searching "Pressure" to find all pressure-related equipment
Equip. No. Unique numerical identifier (e.g., 999, 998) Click header to sort numerically (3-digit IDs for main equipment)
CL (Classification) Equipment category (e.g., "Helium Leak Detector") Identifies equipment type at a glance
ID. Company asset number (e.g., BGT/PU/PRV 109) Official identifier used in documentation
S/No. R/No. Serial/Reference number field Displays manufacturer serial numbers when available
Description Detailed specs (e.g., "4 Inch X 20m X 40 Bar") Contains exact model numbers and technical details
Pagination Navigation through 150+ pages of equipment Use numbered buttons or arrows to browse pages

Equipment List Navigation Tips

  • Search by equipment type (e.g., "Hoses") to see all variants (12 Inch, 8 Inch, etc.)
  • Note the BGT/ prefix in ID numbers indicates company asset numbering system
  • Pressure equipment shows detailed specs (e.g., "Sn 15rvp9072ce") for precise identification
  • Hoses include size, length and pressure ratings in descriptions

For additional assistance with equipment lookups, contact the GEMRS support team.
