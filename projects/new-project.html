Creating a New Project

When creating a new project, you'll need to fill in the following key information:

  1. Project Scope: Select either Mobilization or Re-Mobilization to define your project's scope.
  2. Project Details: Enter essential information including:
    • Project Code (e.g., PRJ000816)
    • Client Company (e.g., SPDC, NAQC, NPDC, NNPC)
    • Project Title (e.g., 24inch x 32xm Cphale - Borny Flushing)
    • Department (select from dropdown)
  3. Location Information: Specify the project location details including:
    • Address lines
    • Location
    • WELI
    • Pipeline
    • Terrain (select from dropdown)
    • District, Station, and Field
  4. Project Team: Assign key personnel including:
    • Project Coordinator
    • Project Engineer
    • Project Manager
    • Focal Person
  5. Dates: Enter the Mobilization (Mob Date) and Demobilization (Demob Date) dates.
  6. Project Status: Set the current status (defaults to PREPARATORY).
  7. Remarks: Add any additional notes or comments about the project.
  8. Add Project Button: After completing all required fields (marked with *) and optional information:
    • Click this button to submit and save your new project
    • The system will validate all entered information
    • Upon successful submission, you'll receive confirmation and be redirected to the project view
    • The project will be logged with your name as the creator and current date

Pro Tip: Complete all required fields (marked with *) to ensure proper project documentation.

Project Log and Tracking

The system automatically maintains a project log that tracks:

  1. Creation Details: Records who created the project and when (e.g., "Added by: Unye-Awaji Greatman Date: 17/04/2025").
  2. Status Changes: Tracks any updates to the project status throughout its lifecycle.
  3. Team Assignments: Records changes in project team members and their roles.
  4. Date Updates: Logs any modifications to key project dates.

Note: Ensure all project information is accurate before submitting, as this forms the basis for project tracking and reporting.
