GLOBAL MATERIAL REQUEST/ISSUE SLIP (GMRIS)

The standardized enterprise-wide system for all material requests and issues within B.G. Technical Limited facilities and project sites. GMRIS ensures proper documentation, tracking, and accountability for all material movements throughout their operational lifecycle.

Navigating the GMRIS Slip

  • FORM: Standard material slip identifier (e.g., 00)
  • SLIP No: Unique tracking number for each material request (e.g., 12739)
  • PO No.1: Primary purchase order reference for the material
  • COST CENTRE: Financial accountability code for the request

Field Description Example

YORKOR CODE: Project/Department coding To be completed
BRIDGET CODE: Material classification code To be completed
CATEGORY/CANALETE: Material category description To be completed

Remarks Section

The GMRIS slip includes an extensive remarks section with 331 remark fields for detailed documentation:

  • Use remarks for special instructions, conditions, or exceptions
  • Numbered sequentially from REMARK 1 to REMARK 331
  • Include material specifications, handling requirements, or delivery notes
  • Reference related documents or previous requests when applicable

GMRIS Authorization Workflow

Approval Signatures

  • REQUESTED BY: Department Representative
  • VERIFIED BY: Technical Supervisor
  • AUTHORISED BY: Department Head
  • PROCESSED BY: Materials Management

Status Tracking

  • REQUEST STATUS: Draft/Submitted/Processing
  • VERIFICATION STATUS: Pending/Completed
  • AUTHORIZATION STATUS: Pending/Approved
  • ISSUE STATUS: Pending/Fulfilled

GMRIS Slip Best Practices

  • Complete all mandatory fields including FORM number and SLIP No
  • Reference the correct PO number when applicable
  • Use specific YORKOR and BRIDGET codes for proper tracking
  • Provide detailed material descriptions in CATEGORY/CANALETE field
  • Utilize remarks section for special instructions or conditions
