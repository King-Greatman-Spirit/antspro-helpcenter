Understanding the Link SHO Gun Stock Tracker

Current Status: No Stock Data Available

The system shows zero recorded transactions for this item (Code: 5359).

Table Columns Explained

  1. Date: Transaction date (currently empty)
  2. Ref: GRIMGR & GMR18:
    • GRIMGR: Goods Receipt/Issue Movement Reference
    • GMR18: General Material Request form ID
  3. Qty Received: Units added to inventory
  4. Qty Issued: Units withdrawn from stock
  5. Qty Damaged/Shortage: Lost/wasted units
  6. Balance: Current available quantity
  7. Unit: Measurement (e.g., Pc for pieces)
  8. Stock Card: Historical transaction ledger
