Navigating the Inventory Management Interface

  • Dashboard: Use this button to return to the main dashboard view from any section.
  • Back: Takes you to the previously visited page or section.
  • Refresh: Reloads the current page to display the latest updates or changes.

Navigating and Using the Category & Document Tables

The page is divided into two major sections:

  1. Store Items Categories: A categorized list of all warehouse inventory items, including remarks and category view options.
  2. Store/Warehouse Request, Return & Delivery Vouchers: A summarized table of key document types used in warehouse management (like Return Slips, Waybills, etc.).

Explaining the Store Items Categories Table

  1. #: Serial number of each inventory category (e.g., 57 to 66).
  2. Category: The name of the inventory category (e.g., WELDING MATERIALS, VEHICLE TYRES).
  3. Remarks: Brief description, status, or notes related to each category (editable inline).
  4. View Category: A button üëÅÔ∏è View that lets you open and see detailed items under each category.

Table Features & Controls

  1. Search Bar: Located at the top-right of the table, use it to quickly find specific categories.
  2. Records per Page: Lets you control how many rows to display per page (e.g., 10).
  3. Pagination: Located at the bottom right; lets you navigate through multiple pages of inventory categories.
  4. Sortable Headers: Click on column headers to sort the data ascending/descending.

Store/Warehouse Request, Return & Delivery Vouchers

This second section of the interface lists various warehouse documentation tools for inventory management.

  1. #: The index number for each document type.
  2. Document: Type of document, such as Material Request Slips, Waybills, etc.
  3. Posts: Icon that links to available posts or records under that document type.
  4. Links: The üëÅÔ∏è View button lets users open and manage the respective documents.

Tip: Click the "View" button on each row to see more details, edit remarks, or manage item entries directly.
