ANTSPRO II (HR) Leave Help Center

The HR environment of ANTSPRO II is designed to streamline employee management processes, providing tools for tracking staff information, leave approvals, and department-specific details. This guide will help users navigate the HR dashboard effectively.

Navigating the HR Dashboard

Dashboard Features

  1. Top Navigation Bar:
    • Includes tabs such as Accounts , My Profile , Leave , Password , Back and Refresh.
    • Use these options to view accounts related information, update personal information, manage leave requests, reset passwords, go back or refresh the page.
  2. Side Menu:
    • Contains links to Home , Anniversary , Support , and Logout
    • These options facilitate quick navigation to general features and support resources.

Leave Applications Interface Overview

Key Functionalities

  • New Application: Initiate a fresh leave application.
  • Completed (Archive): Access all previously approved and archived leave applications for reference.
  • Pending HR Approval: Lists applications awaiting HR review and confirmation.
  • Pending Mgrs. Approval: Displays applications pending managerial approval.
  • Pending Authorisation: Shows applications waiting for final authorization.
  • Pending Submission: Applications that have been started but not yet submitted.
  • Leave Applications Archive: Repository of all past leave applications for historical reference.

Leave Applications Table Structure

  • S/N: Serial Number of the application.
  • Date: The date the application was submitted.
  • Department: The department of the employee requesting leave.
  • Type: The type of leave (e.g., Annual, Sick, Maternity).
  • Authorization: Current authorization status of the leave application.
  • Manager's Approval: Indicates if the application has been approved by the manager.
  • HR Approval: Shows whether HR has given approval.
  • Log: Provides access to the application history and updates.
  • Letters: Access generated leave letters associated with the application.
  • Forms: Links to the completed application form for detailed view.

Tips for Efficient Use

  • Regularly refresh the page using the Refresh button to ensure updated data visibility.
  • Use Ctrl + F (or Cmd + F on Mac) to search for specific staff information quickly.
  • Contact support via the provided link for technical issues or inquiries.
