Receiving Workflow

Approval Signatures

  • RECEIVED BY: Warehouse Personnel
  • VERIFIED BY: Inventory Supervisor
  • AUTHORISED BY: Department Head
  • ACKNOWLEDGED BY: Project Manager

Status Tracking

  • RECV. STATUS: Pending/Received
  • VER. STATUS: Pending/Verified
  • AUTH. STATUS: Pending/Approved
  • ACKN. STATUS: Pending/Confirmed

Delivery Workflow Stages

Understanding the OPDN Workflow

  1. Pending Receipt: Shipment arrived but not yet processed
  2. Pending Verification: Requires physical inspection
  3. Pending Authorization: Needs department approval
  4. Pending Acknowledgement: Awaiting final project confirmation
  5. Completed: Fully processed delivery notes

Key Features

  • 20-line item capacity per delivery
  • Multiple remark fields for receiving notes
  • Actual vs expected quantity comparison
  • Full audit trail of all receiving actions
