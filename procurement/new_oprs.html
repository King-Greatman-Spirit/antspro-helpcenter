Procurement Workflow Stages

Understanding the OPRS Workflow

  1. Pending Submission: Draft requests not yet formally submitted
  2. Pending Authorisation: Requires department head approval
  3. Pending Approval: Needs procurement manager sign-off
  4. Pending Dispatch: Approved requests awaiting shipment
  5. Pending Acknowledgement: Shipped items awaiting receipt confirmation
  6. Completed: Fully processed procurement requests

Key Features

  • 20-line item capacity per request
  • Multiple remark fields for special instructions
  • Expected date tracking for each item
  • Full audit trail of all actions
