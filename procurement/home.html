Overseas Procurement Overview Help Center

Overseas Procurement in Antspro is categorized into two distinct but connected modules (Overseas Procurement Request Slips, Overseas Procurement Delivery Notes) to streamline the overseas procurement workflow and ensure accountability at every stage.

Tabs Explained

Each tab on the Overseas Procurement page filters records based on their current status in the procurement workflow:

  1. Completed: Displays procurement entries that have gone through the full workflow — from submission to acknowledgement. These are finalized and available for reference only.
  2. Pending Acknowledgement: Lists procurement records where items have been received or dispatched but are still awaiting formal confirmation or acknowledgement by the relevant personnel.
  3. Pending Dispatch: Shows approved requests that are yet to be dispatched. This stage typically follows approval and authorization and precedes delivery.
  4. Pending Approval: Contains procurement requests that have been submitted and authorized but are still awaiting final approval from the designated approver or project head.
  5. Pending Authorisation: Displays procurement requests that are pending initial authorization, usually from a supervisor or department head, before moving forward in the workflow.
  6. Pending Submission: Includes drafts or requests that have been created but not yet officially submitted into the procurement workflow. These may still be undergoing edits or internal reviews.

Overseas Procurement Request Slips

This module displays procurement requests initiated for overseas supplies. Each request goes through the following key stages:

  1. Date of Request
  2. Requisition Number
  3. Project & Requesting Officer
  4. Authorisation
  5. Approval
  6. Dispatch
  7. Acknowledgement
  8. Edit

Overseas Procurement Delivery

This module manages the delivery tracking of items related to overseas procurement. Each record captures:

  1. Date of Delivery
  2. Requisition Number
  3. Project & Requesting Officer
  4. Status of Submission, Dispatch, Reception, and Acknowledgement
  5. Edit

Navigating and Using the Table Efficiently

To make the most of each of the Overseas Procurement table, the platform provides multiple user-friendly tools to help you find, view, and analyze records quickly:

  1. Search Input: Located above the top-right corner of the table, this feature allows you to instantly filter results by typing keywords such as Req. No, Project, Requesting Officer, or any relevant value. The results are updated in real time as you type.
  2. Records per Page Dropdown: Found on the top-left of the table, this dropdown allows you to control how many entries are visible at once. You can expand the view from 10 to 25, 50, or more records per page for broader visibility.
  3. Pagination Controls: Below the table, you’ll find numbered buttons (1, 2, 3…) that let you navigate between pages. Simply click on any page number or use “Previous” and “Next” to move through the records.
  4. Sortable Columns: Each table header has a small arrow icon. Clicking on these arrows allows you to sort the table data in either ascending or descending order based on that column (e.g., by Date, Net Amount, Vendor, etc.).

Pro Tip: These tools are designed to streamline your workflow, helping you access and manage large datasets with ease and precision.
