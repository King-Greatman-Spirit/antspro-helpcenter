Navigating and Using the Equipment View Table

The Equipment View Table includes multiple features for filtering, sorting, and reviewing equipment data efficiently:

  1. Search Bar: Located in the top-right, this allows you to filter the table by typing keywords like Equipment Name, Type, Status, etc.
  2. Records per Page: Control how many items are shown per page (e.g., 10, 25, 50) for easier browsing.
  3. Pagination: Navigate through multiple pages using the numbered buttons or "Next"/"Previous" controls.
  4. Sortable Headers: Click on column headers to sort the data ascending/descending (e.g., by Quantity, Status, or Category).

Pro Tip: Use the search and sorting functions together for rapid access to specific equipment data.

Understanding the Equipment View Table Layout

Here’s a breakdown of what you’ll find in the Equipment View Table:

  1. Serial No.: Indicates the record’s position in the table.
  2. Category: General classification of the equipment (e.g., Accounts Vouchers).
  3. Description: A brief explanation of the item.
  4. BGT ID No.: Unique budget tracking ID.
  5. Registration No.: Assigned registration number (if available).
  6. Issue Date: When the asset was issued.
  7. Return Date: Expected or recorded return date.
  8. From / To: Indicates transfer or movement history of the equipment.
  9. Data Sheets: Links or views to attached documents or metadata.

Tip: Use the search bar and records-per-page dropdown to quickly locate items.

Always ensure data is entered accurately to maintain reliable equipment logs.
