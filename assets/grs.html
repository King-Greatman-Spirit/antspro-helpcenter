Return Workflow Stages

Equipment returns progress through these verification stages:

  1. Pending Submission: Initial stage where returns can be edited before formal processing.
  2. Pending Authorisation: Requires department approval. View/print available but no edits.
  3. Pending Approval: Awaits final verification from equipment managers.
  4. Pending Reception: Approved returns awaiting physical receipt. Receive action available.
  5. Received: Completed returns with equipment verified. View/print for records.

Workflow Best Practices

  • Edit only for Pending Submission returns
  • Click Receive when physically accepting equipment
  • View/Print signed slips for physical records
  • Verify equipment condition before approval

Important Notes

  • All returns require triple verification (Sign/Authorize/Approve)
  • Editing disabled after submission
  • Contact Warehouse for damaged equipment
