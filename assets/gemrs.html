Request Status Workflow

Equipment requests progress through these stages:

  1. Pending Submission: Initial draft stage where requests can be edited before formal submission.
  2. Pending Authorization: Requires department head approval. Requests can be viewed/printed but not edited.
  3. Pending Approval: Awaits final sign-off from equipment managers. View/print available.
  4. Pending Dispatch: Approved requests awaiting equipment issuance. Issue action available.
  5. Dispatched: Completed requests with equipment released. View/print for records.

Workflow Tips

  • Edit requests only in Pending Submission stage
  • Use View/Print for authorization/approval reference
  • Click Issue when physically dispatching equipment

Important Notes

  • All dispatched requests are automatically archived
  • Editing disabled after submission to maintain audit trail
