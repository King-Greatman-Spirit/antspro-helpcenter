ANTSPRO II Operations Help Center

The Operations environment of ANTSPRO II is designed to streamline operational processes, providing tools for managing notifications, assets, warehouse, business development, projects, procurement, work order, W..C.C., Waybills, ICT Complaints and more. This guide will help users navigate the Operations dashboard effectively.

Navigating the Operations Dashboard

Dashboard Features

  1. Side Menu:
    • Contains links to Assets, Warehouse, Buiz. Dev, Projects, Procurement, Work Order, W.C.C., Waybills, ICT Complaints, Support, Help, and Logout.
    • These options facilitate quick navigation to operational features and support resources.

Notifications Section

  1. The central area displays detailed notification information
  2. Columns include:
    • Search per page: Filter and locate specific notifications
    • Subject: Subject of notifications.
    • Document Number: Reference number of notifications.
    • Submitted: Submission date of notifications.
    • Open/View: Expand notification for full context.

Safety & Compliance Protocols:

  1. All operations must comply with Shell's 32 U.S. Survey Rules
  2. Mandatory ISO 9001:2015 quality management adherence
  3. Required PPE usage in designated work areas
  4. Authorization needed for safety-critical equipment access

Key Functionalities

  • The Operations module allows users to track notifications, manage assets, oversee warehouse operations, track business development activities, manage projects, handle procurement processes, manage work orders, oversee W.C.C. (if applicable), manage waybills, handle ICT complaints.

How to Use This Page

  1. Track Notifications:
    • Monitor and manage notifications in the notification section.
  2. Manage Assets:
    • Use the Assets link in the side menu to manage company assets.
  3. Access Support Resources:
    • Click on the Support link in the side menu for assistance.
  4. Logout Securely:
    • Use the Logout option once tasks are completed to ensure account security.

Standard Operational Workflows

  • Notification Handling: New → Reviewed → Actioned → Archived
  • Procurement Process: Request → Approval → PO → Delivery
  • Work Order Lifecycle: Creation → Assignment → Completion → Sign-off
